## 👋 About

I'm an Electronics & Computer Engineering student at Thapar Institute of Engineering & Technology, building full-stack apps and exploring AI agents and LLM-powered tools. Open to Data Analyst, Web Development, and SDE opportunities.

## ✨ Features

- Responsive, single-page design with smooth scroll navigation
- Scroll-triggered reveal animations using the Intersection Observer API
- Animated stat counters
- Live-typing demo for the InsightAI project section
- Clean dark theme with accent highlights

## 🛠️ Tech Stack

**Frontend:** HTML5, CSS3, JavaScript (vanilla)
**Fonts:** Space Grotesk, Inter (Google Fonts)

## 💻 Running Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR-USERNAME/portfolio.git
   ```
2. Open `index.html` in your browser, or run it with a local server (e.g. VS Code's Live Server extension).
